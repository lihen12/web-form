Responsive Web Design

What problems does Responsive Web Design tackle?

Responsive Web Design (RWD) aims to make pages look great at any resolution on all devices. There are so many devices in this day, that it’s impossible to have separate html files for every single resolution and device. In the case of RWD, pages use “the same URL and the same code wheter the use is on a desktop computer, tablet, or mobile phone - only the display adjusts or ‘responds’ according to the screen size” (https://developers.google.com/search/mobile-sites/get-started). RWD is being a commonly accepted practice when thinking about the design and structure of websites. Market research has been done that shows that the number of visitors on web pages that utilize RWD to make pages easier to read increased per visit. To start things off, a Mobile First approach is a good way to start thinking about RWD since the majority of the world uses mobile smart phones to do the majority of their internet activity.
